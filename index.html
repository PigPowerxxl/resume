一种基于GAN的矢量数据到瓦片地图智能生成模型

An Intelligent Tile-Map Generation Model based on Vector Data by GAN

  • Research stage : Under Experiment
  • Key Words : Map Generation; Network Tile-Map; Deep Generation Model; Image Translation; CycleGAN

         研究旨在基于深度学习模型中生成对抗网络结构实现由矢量数据自动化生成瓦片地图。现有研究首先对图片生成领域、地图生成领域国内外研究进行了综述，在此基础上提出目前地图自动化生成领域的主要研究问题与不足，并针对性提出一种新的复合双层生成对抗网络结构，实现由结构化数据到栅格数据的地图生成模型，通过矢量数据到栅格数据进行实验验证。

        本文的研究内容主要包括：
                （1）构建基于矢量数据输入的条件生成对抗网络的瓦片地图自动生成模型生成对抗网络已被大量研究证实可以运用在图像生成、图像翻译以及地图生成领域，同时也是目前地图自动生成领域所使用的主流方法。为了保证所生成的地图的质量，现有研究大多以引入信息辅助来优化模型，但鲜有模型就单层条件生成对抗网络的结构提出改进。为此，本研究提出基于矢量数据输入的双层生成对抗网络模型，综合引入矢量数据的图形信息与属性结构信息来进行瓦片地图的自动生成，拟实现对生成的地图质量的提升。
                （2） 引入矢量数据属性信息优化模型以提升生成地图的真实性与可用性现有研究中，基于生成对抗网络的模型生成的地图难以解决缺乏真实性与可用性的问题，通常会出现道路间断、道路不等宽和建筑物与水系压盖等情况，同时地图需要能够准确使用具有指代意义的颜色对不同类别地物进行赋色（如水系一般采用蓝色系等），而现有方法并没有从根本上解决这些问题，这反应了模型缺乏对地图的地理属性与拓扑属性的学习能力；同时现有研究对生成地图的质量（尤其是地理属性与拓扑属性）也缺乏量化评价，因此这一部分的研究通过在上一步研究中提出的瓦片地图自动生成模型中引入属性信息以优化模型，在双层生成对抗网络的结构中分别引入图形信息与属性信息，拟构建能够生成高质量瓦片地图的模型，并通过构建特定指标，与主流地图生成模型进行量化比对，评价本研究模型生成地图的质量。

顾及几何特征与功能特征的建筑物分级显示策略

Hierarchical Display Strategy of Buildings Considering Geometric and Functional Features

  • Research stage : revision
  • Key Words : 3D map; Multilevel Mapping; Information Entropy;

          城市区域三维地图虽具有较好的仿真性，但也会带来较强的渲染负担，从信息传输的角度来说，这种不分重点的展示方式并不利于关键信息的高效获取，提高了地图认知成本。针对上述问题，本文提出一种建筑物分级评估模型，引入几何特征属性与社会功能属性特征来量化评估建筑物的可视等级，以此提出基于特征加权信息熵的多级映射策略，将可视等级结果映射于多级地图上，最后通过实验验证其可用性，可为改进三维地图的认知效能提供参考。
